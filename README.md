# UDTheoretizer: repository of the project
This is a repository of the module called UDtheoretizer. The module serves for reversing some syntactic dependencies to make the syntactic parsing more theoretically accurate.

In the repository, you can find all the materials of the project - the code, the demonstration of the usage, but also a development notebook with tests. The following files are present:
- all the code of the module written in Python (files 'UDTheoretizer.py', 'inverters.py', 'visualizers.py')
- a notebook with a demonstration of how and for what purposes the module can be used (file 'UDTheoretizer_demo.ipynb')
- a development notebook with the tests for the Russian language (file 'UDTheoretizer_dev.ipynb')

The module is currently under development, so all comments and questions are very welcome!

Also, the module has not yet been published as a Python library, but you can use it with

```
!git clone https://github.com/IrinaPolitova/UDTheoretizer
from UDTheoretizer.UDTheoretizer import UDTheoretizer
```

... or other means.
